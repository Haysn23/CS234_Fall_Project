To run this application please first unzip the JPHandlerRun folder. Move the JPHData file to this location "C:\Users\Public\Documents" when file is in correct place the path will be "C:\Users\Public\Documents\JPHData". Once this is complete please open the dist file. 
Click on the JobPostProject.jar file to run this application. 
To run the project from the command line, go to the dist folder and
type the following:

java -jar "JobPostProject.jar"
